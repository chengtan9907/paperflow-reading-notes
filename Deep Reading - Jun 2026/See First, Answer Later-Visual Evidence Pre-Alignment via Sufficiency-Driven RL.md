# See First, Answer Later: Visual Evidence Pre-Alignment via Sufficiency-Driven RL

> ★★★☆☆ 值得快速浏览 · 模型 openai-compatible/deepseek-v4-flash

🏷 关键词：visual evidence pre-alignment · multimodal large language models · reinforcement learning · group relative policy optimization

## 一句话总结

VEPA通过在预训练与后训练之间插入基于充分性驱动的GRPO优化问题条件视觉证据生成的中间阶段，显著增强了多模态大语言模型对细粒度视觉信息的利用与鲁棒性。

## 摘要

> 多模态大语言模型（MLLMs）在视觉输入与文本推理之间常存在不一致，标准的两阶段训练（大规模图文预训练+监督微调与强化学习）导致模型依赖语言先验而忽视细粒度视觉证据。本文提出视觉证据预对齐（VEPA）作为中间阶段，在预训练后、后训练前引入充分性驱动的GRPO（Group Relative Policy Optimization），让模型生成问题条件化的视觉描述，再由冻结的纯文本LLM（盲读器）基于该描述回答并验证描述是否足以恢复正确答案。通过这种“先看后答”的预对齐，模型被迫习得与问题紧密相关的、充分且精准的视觉证据提取能力。

Q1: 这篇论文试图解决什么问题？

当前多模态大语言模型（MLLMs）的标准训练流水线（大规模基于caption的预训练→SFT/RL后训练）存在根本性缺陷：预训练阶段使用的短粗描述（short coarse captions）将模型偏差指向显著物体，而对完成特定问答任务所需的细粒度视觉证据（如物体数量、空间关系、具体属性）关注不足。这导致下游任务中模型倾向于通过语言先验（language priors）猜测答案，而非真正依赖视觉信号——例如，在仅提供问题而不给图像时，模型仍能取得可观准确率。这种“盲答”行为在需要精确视觉推理的基准（如图表问答、细粒度计数、空间推理）上导致系统性失败。论文直接指出了该范式的问题：模型学到的视觉对齐是弱且不充分的，未能建立任务驱动的视觉注意力机制。因此，需要一种机制在进入后训练（instruction following / reasoning）之前，先强制模型学习提取问题相关的、充分的视觉证据，并确保该证据足以让一个纯文本推理器（盲读器）还原正确答案。

Q2: 有哪些相关研究？

相关工作涵盖三个线索：1）MLLMs的训练范式：以LLaVA、BLIP-2、InstructBLIP为代表的caption预训练+指令微调框架，其弱点已被一些工作指出，如MMVP、MMBench等评测中暴露的对细粒度视觉的忽视；2）强化学习在MLLMs中的应用：RLHF、DPO以及最近用于视觉任务的RL方法（如LLaVA-RL、DPO-V）尝试对齐人类偏好，但通常聚焦于后训练阶段，且奖励设计多针对答案正确性而非视觉证据的充分性；3）视觉锚定与注意力引导：一些工作通过显式注意力监督（如attention map对齐）或硬注意力掩码来强制模型关注特定区域，但这些方法需要区域标注或额外的视觉定位模型。VEPA的独特之处在于：它利用一个冻结的纯文本LLM作为“盲读器”，通过问题的答案是否正确来隐式监督视觉描述的质量，从而在没有细粒度标注的情况下实现任务导向的视觉证据提取。这与“可解释视觉问答”中生成格点证据（grounding）的思路类似，但VEPA更强调证据本身的信息充分性而非定位精度。

Q3: 论文如何解决这个问题？

VEPA是一个插入在标准预训练（第一阶段）与后训练（第二阶段）之间的干预阶段。其核心流程如下：
1. 给定一个图像I和关联问题Q，MLLM（当前训练中的视觉-语言模型）首先生成一个问题条件化的视觉证据描述E（question-conditioned visual evidence），例如“图片中红色汽车旁边有3个停车标志”。
2. 该描述E与原始问题Q被送入一个冻结的纯文本大模型（盲读器，例如Qwen2.5-7B-Instruct），盲读器基于E和Q产生答案A'。
3. 奖励由两部分组成：(i) 充分性奖励：答案A'与真实答案A的准确匹配程度（通过规则或判断模型打分）；(ii) 长度惩罚：鼓励描述简洁，避免冗长但无用的信息（因为盲读器本身具备推理能力，过长的描述可能包含冗余噪声）。
4. 使用GRPO（Group Relative Policy Optimization）优化MLLM：对同一问题生成一组（group）若干候选描述，计算每条的相对奖励优势，更新MLLM参数以增加高奖励描述的生成概率。GRPO的组相对性质使得模型不需要显式优势估计，而是通过组内比较自动平衡充分性与简洁性。
5. 关键设计：盲读器完全冻结，因此训练信号完全来自“视觉描述能否使纯文本推理恢复正确答案”，这迫使MLLM提取与问题直接相关的、不冗余但足够完整的视觉信息。VEPA使用与下游任务相同的训练数据集（如A-OKVQA）中的问答对，但不使用答案作为直接监督，而是通过盲读器间接提供。
6. 经过VEPA训练后，MLLM再进入标准SFT/RL后训练阶段，此时模型已经具备更敏锐的视觉感知基础。

Q4: 论文做了哪些实验？

实验在多个视觉问答基准上进行，包括：ChartQA（图表推理）、GQA（场景细节）、A-OKVQA（知识型问答）、TextVQA（文本密集型）、MMStar（多维能力评测）。主要实验设置：
- 基模型：使用LLaVA-1.5架构（7B/13B），预训练后直接进入标准SFT的模型作为基准；VEPA+SFT模型：先经过VEPA训练（使用A-OKVQA的问答对），再与基准进行相同SFT训练。
- 盲读器变体：测试不同规模（如3B、7B）的LLM（Qwen2.5系列），以及不同指令模板下的鲁棒性。
- 对比消融：移除VEPA、替换奖励函数（纯准确率、仅长度惩罚）、替换优化器（PPO vs GRPO）、改变描述长度限制。
- 反事实鲁棒性测试：在推理时对图像施加高斯模糊、局部噪声、纯噪声三种退化，计算准确率保持率（corrupt accuracy / clean accuracy）和AUC，衡量模型对视觉输入的依赖程度。
- 额外分析：问题仅（question-only）准确率比较，即在无图像输入下模型的随机猜测基线，用于量化语言先验利用程度。

Q5: 发现了什么实验现象？

1. 主基准提升：在ChartQA上，VEPA+SFT相比标准SFT准确率从51.88%提升至58.96%（+7.08%）；在GQA上从52.13%提升至53.07%（+0.94%）。提升幅度因任务而异，图表推理受益最大，场景细节次之。
2. 反事实鲁棒性：在图像退化下，VEPA模型表现出更低的保持率（即clean与corrupt准确率差距更大），但在偏噪声条件下保持率更高，说明模型更依赖有效的视觉信息而非退化后的误导信号。AUC综合指标上VEPA显著低于基准，表明其视觉依赖更强。
3. 问题仅（question-only）准确率：标准SFT在部分任务上展现出较高的问题仅准确率（如ChartQA上超过42%），表明模型学会了利用语言偏差。VEPA显著降低了问题仅准确率（例如ChartQA从44.27%降至42.53%），但同时提高了真实准确率，说明模型转而更依赖视觉输入。
4. 描述长度与充分性：VEPA训练出的描述平均长度比基线caption更短（如从464.21降至365.16），但蕴含的信息足以使盲读器恢复正确答案，证明“简洁而充分”的目标被有效优化。
5. 盲读器规模影响：使用7B盲读器比3B盲读器带来的下游增益更大，且训练更稳定；但即使使用3B盲读器，VEPA仍优于基准。
6. 消融实验：移除GRPO的组相对奖励或替换为绝对奖励会导致性能下降；增加长度惩罚权重需要谨慎平衡，过强会阻碍信息覆盖。

Q6: 有什么可以进一步探索的点？

1. 扩展至更复杂的推理任务：当前VEPA主要集中在单轮事实性问答，未来可探索多步推理（如CoT）或对话场景中视觉证据的递进式预对齐。
2. 盲读器与MLLM的联合适应：探索在VEPA后阶段微解冻盲读器部分层，以允许视觉描述与推理能力协同进化，但需防止先验偏见重新侵入。
3. 多尺度证据生成：将VEPA扩展至区域级与像素级描述（如边界框或分割mask），使证据不仅限于自然语言，也可支持可解释定位。
4. 跨域泛化：VEPA在当前数据集训练，其在全新域（如医学影像、遥感）上的迁移效果需进一步验证。
5. 更高效的奖励设计：研究自生成答案与真实答案的评分（可学习评分器）替代固定规则，实现更细粒度的充分性度量。
6. 结合检索增强：在生成描述前允许模型检索外部知识，使视觉证据与常识融合。

Q7: 总结一下论文的主要内容

论文提出了VEPA（Visual Evidence Pre-Alignment），一种插入在多模态大语言模型预训练与后训练之间的中间训练阶段，旨在解决标准训练范式中视觉对齐不足导致的“盲答”问题。VEPA的核心创新是使用充分性驱动的GRPO（Group Relative Policy Optimization），迫使MLLM生成问题条件化的视觉描述，并通过一个冻结的纯文本盲读器来验证该描述是否足以让推理模型恢复正确答案。这种“先见后答”的机制无需细粒度视觉标注，仅利用问答对隐式地监督视觉证据提取。实验在多个基准（ChartQA、GQA、A-OKVQA等）上显示VEPA能持续提升准确率，尤其在需要精确视觉理解的图表任务上改善显著（+7%）。反事实评估和问题仅分析证实VEPA模型确实增强了对视觉输入的依赖，减少了语言先验偏差。消融和鲁棒性实验进一步表明VEPA训练出的描述更简洁、信息密度更高。该工作为MLLMs的训练流水线提供了一个简单但有效的干预点，强调视觉感知基础对于后续推理的关键作用。
